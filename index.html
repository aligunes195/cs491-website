      WhosePic: A Face Retrieval Mobile Application

    Most people have an excessive number of photos in their mobile phone. Since we started to use as a camera to our smartphones, almost all of our photos are saved in our phones. Besides, big amount of these photos come from social media platforms or WhatsApp. Tracing all of these photos is nearly imposibble, and it is hard to find the relevant photos among these thousands of photos.

    There are some mobile applications for Android which perform a prediction understand which objects the photo included. However, there is no application to classify the photos based on people in address book of the phone. Therefore, we target to classify photos by human faces in it.

    Our application will be an Android based mobile application that only requests one photo for a person in address book, then it will find out the other photos of this person exist in the memory of smartphone and put these photos together with a user-friendly interface.

    What is our project?

    Our application will be an Android based mobile application that only requests one photo for a person in address book, then it will find out the other photos of this person exist in the memory of smartphone and put these photos together with a user-friendly interface.

    To achieve this, we need to detect one's face, identify all critical points in face of this person which make the person unique, and then we need to compare all of other faces in the other photos in an efficient way. Because this application will be Android based, we need to implement some image processing algorithms into java, which could be challenging.

    Who are we?

    We are Bilkent University students, and this project is our Senior Design Project.
